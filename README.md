# 🌿 GreenRoots Sri Lanka — NGO Website

A responsive and modern NGO website for **GreenRoots Sri Lanka** — built to showcase environmental initiatives, community projects, volunteer opportunities, and contact information.

> ⚡ Designed for real-world usability and accessibility using modern frontend tools.

---

## 🌱 Features

- ✅ Clean, mobile-first responsive design
- ✅ Built with **Vite + React + TypeScript**
- ✅ Styled using **Tailwind CSS** + **DaisyUI**
- ✅ Dynamic components for announcements, contact forms, and statistics
- ✅ Accessible and user-friendly UI
- ✅ Deployment on Vercel

---

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite 5, TypeScript
- **Styling:** Tailwind CSS v4, DaisyUI
- **Icons:** Lucide Icons
- **Deployment:** Vercel

---
